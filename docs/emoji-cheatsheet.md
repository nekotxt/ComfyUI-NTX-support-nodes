Copy/paste any glyph directly into a label string — e.g. `🧪 AILab`.

## Tips
- **Prefix works best:** `🧪 AILab` reads cleaner than a trailing icon.
- **Variation selectors:** emoji with a trailing `️` (⚙️, 🎞️, 🏷️) sometimes render as flat/mono
  glyphs depending on the font. Plain code points (⭐, 🔥, 👑) are the safest cross-platform.
- **One icon per label** — two competing glyphs look noisy in the node list.
